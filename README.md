# 🤝 Әлеуметтік көмек порталы (AleumettikKomek)

Қазақстан Республикасының әлеуметтік көмек түрлері туралы ақпараттандыруға арналған веб-қосымша.

## 📋 Жоба сипаттамасы

Бұл веб-қосымша HTML, CSS және таза JavaScript тілдерінде жазылған. Деректер `localStorage` арқылы сақталады (серверсіз жұмыс істейді).

## ✨ Мүмкіндіктер

- ✅ **9 түрлі әлеуметтік көмек туралы ақпарат беттер** (АӘК, көп балалы отбасы, мүгедектік, зейнетақы, т.б.)
- ✅ **Пайдаланушы тіркелуі** (ЖСН валидациясы, құпия сөз қорғанысы)
- ✅ **Жеке кабинет** (профиль, өтініш тарихы, мәртебесін қадағалау)
- ✅ **Онлайн өтініш беру формасы**
- ✅ **Әкімші панелі** (өтініштерді басқару, мақұлдау/қабылдамау)
- ✅ **Адаптивті дизайн** (мобильді құрылғыларға бейімделген)
- ✅ **Статистика мен сүзгілер**

## 📁 Файлдар құрылымы

```
AleumettikKomek/
├── index.html              ← Басты бет
├── aids.html               ← Көмек түрлерінің тізімі
├── aid-detail.html         ← Әр көмек туралы толық ақпарат
├── apply.html              ← Өтініш беру формасы
├── login.html              ← Кіру беті
├── register.html           ← Тіркелу беті
├── cabinet.html            ← Пайдаланушының жеке кабинеті
├── admin.html              ← Әкімші панелі
├── about.html              ← Байланыс беті
├── css/
│   └── style.css           ← Барлық стильдер
├── js/
│   ├── data.js             ← Көмек түрлерінің деректері
│   └── auth.js             ← Авторизация, өтініштер логикасы
└── images/                 ← Суреттер (қажет болса)
```

## 🚀 Visual Studio-да ашу

### Әдіс 1: Visual Studio Code (ұсынылады)
1. **Visual Studio Code** бағдарламасын ашыңыз
2. **File → Open Folder** → `AleumettikKomek` папкасын таңдаңыз
3. **Live Server** кеңейтпесін орнатыңыз (`Extensions` бөлімінде іздеңіз)
4. `index.html` файлына оң жақ батырмамен басып, **"Open with Live Server"** таңдаңыз
5. Браузерде автоматты түрде ашылады: `http://127.0.0.1:5500`

### Әдіс 2: Visual Studio 2022
1. **File → Open → Web Site...** таңдаңыз
2. `AleumettikKomek` папкасын ашыңыз
3. `index.html` файлын таңдап, **F5** немесе **Ctrl+F5** басыңыз

### Әдіс 3: Тіке ашу
`index.html` файлын браузерде ашыңыз (қосарлы шерту арқылы)

## 📊 Негізгі көмек түрлері (деректер)

1. Атаулы әлеуметтік көмек (АӘК)
2. Көп балалы отбасыларға көмек
3. Мүгедектігі бар адамдарға жәрдемақы
4. Бала туылуына байланысты жәрдемақы
5. Тұрғын үй көмегі
6. Жұмыссыздық бойынша әлеуметтік төлем
7. Жасына байланысты зейнетақы
8. Бала күтімі бойынша жәрдемақы
9. Асыраушысынан айырылған отбасыларға көмек

## 💡 Презентация үшін кеңестер

1. Бастапқыда `admin@komek.kz` / `admin123` арқылы кіріп, функцияны көрсетіңіз
2. Жаңа пайдаланушы тіркеңіз → өтініш беріңіз → әкімшімен мақұлдаңыз
3. Адаптивтілікті көрсету үшін браузерде F12 → Device Toolbar ашыңыз
4. `localStorage` деректерін Developer Tools → Application → Local Storage бөлімінен көруге болады
